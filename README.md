# 📊 Amazon Sales Analysis

## 🔍 Overview
An end-to-end data analysis project on Amazon sales data (128K+ records), focused on transforming raw data into meaningful business insights using Excel, Python, MySQL, and Power BI.

---

## 🛠 Tech Stack
- **Advanced Excel** – Initial data cleaning & exploration  
- **Python (Pandas, NumPy)** – Data preprocessing & analysis  
- **MySQL** – Data storage & query handling  
- **Power BI** – Interactive dashboards & visualization  

---

## ⚙️ Workflow
1. **Data Collection**
   - Imported large dataset (128K+ rows) from kaggle datasets.

2. **Data Cleaning (Excel + Python)**
   - Removed duplicate records  
   - Handled missing & unwanted data  
   - Standardized formats for consistency  

3. **Data Processing (Python)**
   - Performed data transformation  
   - Extracted meaningful patterns & trends  

4. **Database Integration (MySQL)**
   - Loaded cleaned data into MySQL  
   - Executed SQL queries for analysis  

5. **Data Visualization (Power BI)**
   - Built interactive dashboards  
   - Visualized KPIs, trends, and performance metrics  

---

## 📈 Key Insights
- Identified top-performing products & categories  
- Analyzed sales trends over time  
- Evaluated order status & fulfillment patterns  
- Discovered actionable insights for decision-making  

---

## 📊 Dashboard Highlights
- Sales performance overview  
- Category-wise analysis  
- Order & fulfillment insights  
- Trend analysis with interactive filters  

---

## 🚀 Conclusion
This project demonstrates a complete data analysis pipeline—from raw data cleaning to dashboard creation—highlighting practical skills in real-world data handling and visualization.
